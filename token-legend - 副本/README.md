# TOKEN传奇榜 · DeepChart（Vue3 练手项目）

根据《【2026_04_11】Token传奇榜练手》PDF 需求实现的移动端 H5 Demo，包含 **登录页 / 首页 / 排行榜页** 三个核心页面 + **活动弹窗 / 协议弹窗**。

## 技术栈
- Vue 3（`<script setup>`）
- Vue Router 4（Hash 路由）
- Vite 6
- 纯 CSS（移动端手机壳容器，桌面端居中预览）

## 运行
```bash
npm install
npm run dev      # 默认 http://localhost:5173 自动打开
npm run build    # 构建产物到 dist/
```

## 页面与功能对照（PDF 需求）

### 1. 登录页 `/login`
- 密码 / 手机号 / 邮箱 三个 tab（仅密码登录可用）
- 账号密码为**假数据**校验，演示账号：`0001024` / 密码 `123456`（页面有“点此填入”）
- **未勾选协议**点击登录 → 弹窗「登录前，请阅读并同意《用户服务协议》和《用户隐私政策》」（取消 / 同意）
- 点击「同意」→ 自动勾选并继续登录
- 苹果 / Google 等其它按钮 → 跳缺省页
- 校验通过 → 进入首页（记录登录用户数据到 sessionStorage）

### 2. 首页 `/home`
- 顶部红色问候头 + 机器人文案**每 2s 向上滚动**循环（3 句）
- 今日市场概览：**11 个指数**涨跌幅（假数据），点击跳缺省页
- DeepMate：机构猎手卡片**每 3s 向右滚动**（支持手动滑动）+ AI诊股，点击跳缺省页
- 深度探索：**7 个模板**横向滑动 + 查看更多
- 我的自选：TSLA / NVDA / AAPL + 迷你走势
- 机构动向简报 + 「以上数据由 AI 生成，不构成投资建议！」
- 底部搜索框：**最多 20 字**，发送跳缺省页
- 底部导航：首页高亮，其余跳缺省页
- **进入首页（会员）自动弹出「TOKEN传奇榜」活动弹窗**：点「立即参加」→ 排行榜；点其余区域 → 关闭

### 3. 排行榜页 `/leaderboard`
- 左上角「‹ 返回」回首页
- 顶部主标题 + 口号 + **倒计时**（距活动结束 4.24 23:59，实时跳秒）
- 我的训练情况：ID + 名字、当前排名、已消耗 / 总 1000 Token 进度条、当前称号、已解锁权益、**按档位动态引导语**（<200 / 200–500 / 500–800 / ≥800 四档）
- 看看大家的训练情况：参战人数、**前 3 名奖牌**、**高亮当前用户**、按 Token 降序、**>8 名折叠**（展开全部 / 收起榜单）
- 活动说明 + 等级权益表
- **游客**（未登录直接进入）→ 底部展示登录提示条

### 缺省页 `/placeholder`
- 所有「点击跳转至缺省页」的入口统一落地；「‹ 返回」回首页
