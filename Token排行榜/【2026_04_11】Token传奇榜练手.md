# 五、 业务逻辑

### 1、 Token规则

- **所有用户数据赠送 1000 Token 训练包**，**有效期 4月11日 00:00 ～ 4月24日 23:59（共14天）。**

    - 到期未消耗的Token自动清零。

    - 消耗统计：会给出假数据

    - 排行榜按 **消耗Token总量 降序排列**，相同消耗量按更新时间先后排序，时间靠前的排在前面。

### 2、 消耗等级与权益

根据累计消耗Token数量，用户自动获得对应等级称号及权益（实时计算）：

|**等级**|**消耗条件**|**称号**|**权益**|
|---|---|---|---|
|第一级|≥ 200 Token|AI工作流探索者|挂件1 \+ 电子书|
|第二级|≥ 500 Token|AI工作流领航者|挂件2 \+ 200 Token 定额返场卡|
|第三级|≥ 800 Token|AI工作流达人|挂件3 \+ 400 Token 定额返场卡|

注：每达到一个等级，发放对应的权益。

### 3、 排名实时更新

可按程序实现而定，最大单位为24h。
